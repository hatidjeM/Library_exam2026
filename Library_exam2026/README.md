# Library Exam 2026 Project

## Project Overview
This is a Django-based Library web application that allows users to manage **books, authors, and categories**.  
It demonstrates CRUD operations, data validation, custom templates, and styling with Bootstrap.

---

## Features

- **Books**
  - Create, view, edit, delete books
  - Assign authors and categories
  - Validation for published year (must be after 1900)

- **Authors**
  - Create, view, list authors
  - Add biography and birthdate

- **Categories**
  - Create, view, list categories
  - Assign multiple books to a category

- **Front-End**
  - Responsive templates with Bootstrap
  - Cards for books and authors
  - Custom 404 page
  - Navigation menu and footer

- **Advanced**
  - Images for each book
  - Home page shows counts for books, authors, categories

---

## Technologies

- Django 6.0.3
- Python 3.14
- PostgreSQL
- HTML / CSS / Bootstrap
